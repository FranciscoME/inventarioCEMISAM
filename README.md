Este programa es un sistema de control de inventarios realizado en el 2016 para el pequeño hospital donde trabajé, en el cual realiza un CRUD de productos, departamentos y trabajadores. las búsquedas de productos son variadas.
This software is an inventory control system carried out in 2016 for the small hospital where I worked, in which it carries out a CRUD of products, departments and workers. product searches are varied.

Características del sistema:
`*` C#
`*`  Windows Forms
`*` BD: MS SQL Server
`*` Genera reportes en MS Excel.

**ATENCION.- El programa tiene fallos graves de seguridad por lo menos en el login**

**ATTENTION.- The program has serious security bugs, at least in the login**
